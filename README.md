# Analisador de Texto

Programa em Python para análise de arquivos `.txt`: conta palavras, identifica palavras distintas, calcula frequência, busca e substitui termos, entre outras operações.

## Funcionalidades

1. Contar número de palavras
2. Contar palavras distintas
3. Contar número de linhas
4. Exibir frequência das palavras
5. Imprimir linha específica
6. Buscar palavra no texto
7. Substituir palavra e salvar em novo arquivo
8. Abrir outro arquivo `.txt`
9. Encerrar programa

## Como usar

1. Coloque o arquivo `.txt` que deseja analisar na mesma pasta do script.
2. Rode o programa:

```bash
python3 analisador_texto.py
```

3. Digite o nome do arquivo `.txt` quando solicitado.
4. Escolha uma das opções do menu.

## Requisitos

- Python 3
