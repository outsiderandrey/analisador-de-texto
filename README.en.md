# Text Analyzer

Python program for analyzing `.txt` files: counts words, identifies distinct words, calculates frequency, searches and replaces terms, among other operations.

## Features

1. Count number of words
2. Count distinct words
3. Count number of lines
4. Show word frequency
5. Print a specific line
6. Search for a word
7. Replace a word and save to a new file
8. Open another file
9. Exit program

## How to use

The repository already includes a sample `exemplo.txt` file ready to test the program.

1. Place the `.txt` file you want to analyze in the same folder as the script.
2. Run the program:

```bash
python3 analisador_texto.py
```

3. Type the name of the `.txt` file when prompted.
4. Choose one of the menu options.

## Requirements

- Python 3
